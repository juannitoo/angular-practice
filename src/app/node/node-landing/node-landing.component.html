En cours...

Mais j'ai du arrêter dans mon élan et faire du Python pour me préparer pour un hypothétique futur client.
Je reprends peut-être la maintenance d'un robot scrappeur qui renvoie un pdf via mail synthéthisant les résultats obtenus. Alors je me prépare, mon robot d'entrainement est déployé sur une lambda AWS avec selenium sur une image docker dans un registre ECR AWS. Je crée le PDF avec Reportlab. Pour le mail, je n'insiste pas à créer un serveur SMTP, j'utiliserais le leur', je sais déjà que c'est une app Django.

Lorsque je suis revenu sur ce projet quelques semaines plus tard, j'ai préféré m'initier aux tests avec Jasmine plutôt que de me focaliser sur node pour pouvoir postuler à différentes offres d'emplois où l'on utilise autant Node que Java mais aussi quelque fois PHP ou encore C#, mais où l'on fait partout des tests ! J'ai mis en place un pipeline CICD via Github Actions pour ne plus avoir à me soucier de la mise en prod de cette démo. Je n'ai pas complètement terminé mon apprentissage concernant les tests mais je reviens par ici pour avancer un peu et mettre en place les guards et l'authetification, sans pour autant les abandonner.
