JsonPlaceHolder.typicode.com est une api gratuite qui permet de simuler le CRUD. Plusieurs sous-api sont disponibles, j'ai choisi celles des Users. On ne ne peux pas rééllement insérer ou supprimer, mais elle renvoie renvoie l'objet soi disant créé, updaté et supprimé alors c'est suffisant pour m'exercer.

Voilà un exemple de routes possibles :

https://jsonplaceholder.typicode.com/users
https://jsonplaceholder.typicode.com/users/1

Un exemple d'appel en js natif avec l'api posts

fetch('https://jsonplaceholder.typicode.com/posts/1')
.then((response) => response.json())
.then((json) => console.log(json));
 Renvoie : {{ '{' }} id: 1, title: '...', body: '...', userId: 1 {{ '}' }}
